# APS-LAB
This repository contains a collection of Python implementations and analyses for various algorithms covered during the APS Lab sessions. Each lab is documented in a Jupyter Notebook, focusing on algorithm design, complexity analysis, and practical implementation.


File,Description
Lab 1 & 2: Algorithm Basics,Introduction to basic algorithmic logic and Python environment setup.
Lab 3: Growth of Functions,"Analysis of Big O, Omega, and Theta notations with visual plotting."
Lab 4: Searching & Sorting,Implementations of fundamental search (Linear/Binary) and sort algorithms.
Lab 5 - 8: Advanced Topics,"Progressive exercises covering Divide and Conquer, Greedy approaches, and Dynamic Programming."
